Welcome to Buggy

Buggy is a kickstart web application, featuring 99.99% vanilla JavaScript on the client and 100% core Java on the backend. It's a setup for the purists who value simplicity and performance.

Why No Frontend Framework?

Overblown frameworks like Angular or React are just not my thing. Vue is acceptable, but I love the simplicity of vanilla JavaScript. My perfect match? Alpine.js. And for styling? Bulma. Clean, simple, and efficient—what's not to love?

Why Java on the Backend? And Why No Fancy Frameworks?

Let's face it: Spring is a nightmare. The only thing worse than Spring is Spring combined with Hibernate. And the only thing worse than that? Using them together with documentation buried in Atlassian Confluence.
Sure, Node.js or Python backends have their perks. But I'm a Java guy—plain, old, reliable Java. Talk about Lombok or Kotlin, and I might just lose my mind.
